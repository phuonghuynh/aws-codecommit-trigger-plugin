Standard Aws Credentials defined in Jenkins Credentials

The permissions that are required for the credentials you use to read and trigger the job

  • sqs:GetQueueAttributes: Gets attributes for the specified queue
  • sqs:ReceiveMessage: Retrieves one or more messages
  • sqs:DeleteMessage: Deletes the specified message from the specified queue
  • sqs:DeleteMessageBatch: Deletes up to ten messages from the specified queue

Further documentation is available in the AWS SQS Plugin on GitHub.
